## 六、再平衡策略

### 频率

- **推荐：** 每半年一次（6月底、12月底）
- **或：** 每年一次（12月底）

### 触发条件

- 任一资产偏离目标比例 **±5%** 以上

### 操作步骤

1. 计算当前各资产实际比例
2. 对比目标比例，找出偏离资产
3. 卖出超配资产（涨多了）
4. 买入低配资产（跌多了）
5. 恢复到目标均衡比例

### 示例

```
假设目标：沪深300 (18.7%)、中证500 (18.7%)、红利 (18.7%)

半年后实际：
- 沪深300: 22% (超配3.3%)
- 中证500: 17% (低配1.7%)
- 红利: 16% (低配2.7%)

再平衡操作：
- 卖出沪深300: 3.3%
- 买入中证500: 1.7%
- 买入红利: 2.7%

恢复均衡后：各18.7%

```

## 七、核心投资理念

### ✅ 三大优势

1. **四季皆宜：** 无论什么经济环境，都有资产表现好
2. **自动调整：** 通过再平衡实现"低买高卖"
3. **简单易行：** 不需要精准判断经济周期

### ⚠️ 关键要点

- **债券30%** 作为防守底仓，应对强衰退
- **股票70%** 按强复苏、弱复苏、弱衰退均衡分配
- **恒生科技+科创板** 根据市场灵活切换
- **不追涨不杀跌**，严格执行再平衡纪律
- 如果看不清经济周期，**保持均衡配置**本身就能跑赢市场

> "投资不是高考，不要求每道题都会做。不会做就不做，选不出就不选。
